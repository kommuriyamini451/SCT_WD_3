# 🎯 Quiz Game App - SCT_WD_3

Interactive quiz application built as Task 3 for **SkillCraft Technology** Web Development Internship.

### 🔗 Live Demo
**https://kommuriyamini451.github.io/SCT_WD_3/**

## ✨ Features
- **3 Question Types**
    - Single Select - Choose one correct answer
    - Multi Select - Choose multiple correct answers
    - Fill in the Blank - Type your answer
- **Multiple Question Sets** - Auto-loads next set after completion
- **Progress Tracking** - Visual progress bar + question counter
- **Real-time Score** - Live score updates during quiz
- **Detailed Results Page** - Score breakdown with correct/incorrect review
- **Previous/Next Navigation** - Go back and change answers
- **Keyboard Support** - Smooth user experience
- **Dark Neon UI** - Glassmorphism with cyan/purple gradients
- **Fully Responsive** - Works on mobile, tablet & desktop

## 🛠️ Tech Stack
- **HTML5** - Semantic structure
- **CSS3** - Glassmorphism, gradients, animations, Flexbox
- **JavaScript** - Quiz logic, state management, DOM manipulation
